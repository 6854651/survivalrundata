#this app is able to analyse all kinds of survivalrundata
however the app is not able yet to update its database online on its own yet, so the database needs to be manually updated by the creator (hey thats me)
The files to do this are in the file 'Databaseupdater.py'
To get the updated years only modify the year range, if its season 2025/26 use 2025 and 2026 till the summerbreak is accounted for automatically, the next season so 2026/27 use 2026, and it will start automatically from september onwards till 2027 july. You can set Start jaar and end jaar at the same year eg. 2025 and it will run for only 2025/26, this is usefull to update only recent competitions so you dont need to wait ages for all competitions ever to be loaded.
